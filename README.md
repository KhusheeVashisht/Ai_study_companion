# 🎓 AI Study Companion

Turn documents into interactive learning.

🔗 **Live Demo:**
https://aistudycompanion-qkjzcxj5buysi5938rcgrq.streamlit.app/

📂 **GitHub Repository:**
https://github.com/KhusheeVashisht/Ai_study_companion

---

## ✨ Features

✅ Upload PDF / TXT / DOCX

✅ Ask Questions from Documents

✅ Source-backed Answers

✅ Multiple Answer Styles

* Short
* Detailed
* Exam Answer
* Bullet Points
* Simple Explanation

✅ Quiz Generator

✅ Flashcard Generator

✅ Voice Question Input

✅ Export Answers / Quiz / Flashcards

---

## 🧠 Tech Stack

* Python
* Streamlit
* LangChain
* FAISS
* Google Gemini API
* Sentence Transformers

---

## 🚀 Run Locally

```bash
git clone https://github.com/KhusheeVashisht/Ai_study_companion.git
cd Ai_study_companion
pip install -r requirements.txt
streamlit run app.py
```
